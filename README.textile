h1. UriValidator

URI validation for ActiveModel

h2. Installation

<pre>gem "uri_validator"</pre>

h2. Usage

<pre>class Product < ActiveRecord::Base
  validates :video, :uri => true # default protocols are http and https
  validates :manual, :uri => %w(http https ftp)
end</pre>
